# BioAI 日报前端适配需求

## 项目背景
这是一个从通用"AI日报"改造成的"AI+生命科学日报"项目。后端已经完成改造，现在需要前端做相应适配。

## 核心定位
- **主题**：AI + 生命科学（BioAI）交叉领域日报
- **目标用户**：对 AI+生命科学感兴趣的投资人、产品经理、创业者、学生
- **不是**：给生物学家或AI研究员看的专业期刊
- **风格**：专业但亲切，像懂行的朋友分享行业动态

## 域名与部署
- **正式域名**：https://news.aibioo.cn
- **GitHub仓库**：dongyu19920904/BioAI-Daily-Web
- **部署方式**：GitHub Pages（无需更改）

## 后端情况（无需关心细节）
- 后端 API 接口**没有变化**，数据结构与原 AI 日报完全相同
- 后端已配置好 BioAI 相关的 RSS 源和内容筛选逻辑
- 后端 Worker URL: https://cloudflare-bioai-daily.sabrinamisan090.workers.dev

## 需要修改的前端内容

### 1. 品牌相关
- 网站标题/Logo：从"AI日报"改为"BioAI 生命科学日报"或类似
- Favicon：如有需要可更换
- 页脚版权信息

### 2. 页面文案
- 导航栏、首页标语等处的"AI资讯"改为"AI生命科学资讯"
- 任何提及"AI日报"的地方改为"BioAI生命科学日报"

### 3. 主题风格（可选）
- 主题色可考虑偏向生命科学（如蓝绿色系），但保持简洁
- 这是可选的，如果改动成本高可以先不改

### 4. SEO / Meta 标签
- 页面 title、description 等改为 BioAI 相关
- Open Graph 标签更新

### 5. 配置文件
- 检查 hugo.yaml 或其他配置文件中的站点名称、描述等

## 不需要修改的
- API 调用逻辑（后端接口无变化）
- 数据展示结构（后端返回的数据格式相同）
- 部署流程（继续用 GitHub Actions + Pages）

## 建议的工作顺序
1. 先全局搜索"AI日报"、"AI资讯"等关键词，了解需要修改的范围
2. 修改配置文件中的站点名称和描述
3. 修改页面组件中的文案
4. 检查并更新 SEO 相关标签
5. （可选）调整主题色
